# Prepare AE Explorer data for safetyCharts

Prepare AE Explorer data for safetyCharts

## Usage

``` r
MakeAeExplorerData(lData, lMeta = list())
```

## Arguments

- lData:

  A named list containing mapped GSM safety data frames.

- lMeta:

  Workflow metadata containing `domains` mappings.

## Value

A named list with `dm` and `aes` data frames for safetyCharts.
